# MSMods Pro — Termux Enhancement Suite v2.0

**GitHub:** https://github.com/MSMods-Pro  
**Telegram:** https://t.me/MSModsPro  

## Features
- ★ Multi-Design Banner (6 styles, up to 8 words)
- ★ Auto Color Changer (10 themes, changes every Termux open)
- Zsh Setup with Oh-My-Zsh
- Custom Prompt Themes (4 styles)
- Cyber Lock System
- System Info Dashboard
- Network Tools
- File Manager
- Package Manager Hub
- Backup & Restore

## Install
```bash
pkg update && pkg install git -y
git clone https://github.com/MSMods-Pro/Termax-Theme-MSMods.git MSMods
cd MSMods && bash os.sh
```
